Error codes

Description

Errors system and codes description.

In this library the Boost.System's error_code is used. The reason is what it is provided by the Boost.Asio library. So it looks like this:

using error_code = boost::system::error_code;
using system_error = boost::system::system_error;
using error_category = boost::system::error_category;
using error_condition = boost::system::error_condition;

This may be configurable in the future and std::error_code support may be added.
